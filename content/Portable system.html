Portable system

Describe how to prepare a portable Python programming system for Windows 10 64bit system to allow one the maintain CMSiMDE website, Pelican blog and Reveal.js presentation on Github?

影片連結：https://www.youtube.com/watch?v=geKi4BFJb3Y

Python3.7.3升級至Python3.8.2

(1) 在2019fall裡建立cd2020

(2) cd2020內新增data，在data底下新增py32、wscite432、portablegit

(3) 至Python官網找3.8.2版本，拉到最底下點選Windows x86-64  

    executable installer下載安裝檔，路徑設定到data/py382

※選項pip取消勾選

(4) 下載SciTE系統，下載完後解壓縮至data/wsite432
