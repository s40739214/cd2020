week16

IPv6 遠端控制伺服、Ubuntu對外連線設定、連線個人倉儲

1.在VirtualBox勾選2個IPv6選項，主機網路為NAT Network

2.輸入密碼：kmol2020，開啟LXTermial

3.輸入cd /etc→cd apt→cd apt.conf.d→sudo vi proxy.conf

4.密碼為kmol2020，按i編輯，輸入Acquire::http::proxy "http://[2001:288:6004:17::53]:3128";

5.按ESC→輸入:wq儲存

6.sudo apt update 更新apt

7.ifconfig→sudo apt install net-tools

8.cd /etc→cd netplan

9.sudo vi 00-installer-config.yaml，編輯成如影片內容

10.輸入sudo netplan apply更新→ping6 2001:288:6004:17::254測試連線

11.在VirtualBox裡新增IPv6的連接埠號

12.輸入ifconfig查詢IPv6的IP

13.cd cd2020pj1→git config --global --add http.proxy[2001:288:6004:17::53]:3128

14.git pull更新內容，再到tmp下sudo vi oauth_scrum.txt來新增檔案

15.sudo vi wsgi.py，將近端的host改為::，輸入python3 wsgi.py來啟動

16.進入tmp將自己的倉儲clone下來

17.輸入網址https://[2001:288:6004:17:2020:5:0:9214]:19443開啟

影片連結：https://youtu.be/_1OOqh4ibVE
